# Gulp start template

## Browser support

- Chrome
- Edge
- Firefox ESR+
- Internet Explorer 10+
- Safari 8+
- Opera


## Components
**Gulp** build system

## NPM packages

* **jQuery 3**
* **Normalize 8**


## Installation

Clone this repository.

Go to project directory with terminal and run next commands there:
```sh
$ npm install
```

## Configuring
There you can configure some gulp tasks:
**/gulpfile.js**

## Get started
Run command:
```sh
$ gulp watch
```
